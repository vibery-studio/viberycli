# add-authentication-system

Add auth with JWT, OAuth, or Supabase

## Usage

Run this command to execute the specified operation.

## Process

1. Analyze current state
2. Execute required operations
3. Verify results
4. Generate report

## Commands

```bash
# Execute command
npm run add-authentication-system

# With options
npm run add-authentication-system -- --option value
```

## Options

| Option    | Description          |
| --------- | -------------------- |
| --verbose | Show detailed output |
| --dry-run | Preview changes      |

## Output

- Operation results
- Status report
- Recommendations
