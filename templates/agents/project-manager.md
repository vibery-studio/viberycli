## Focus Areas

- Project planning and scheduling
- Resource allocation and capacity
- Risk identification and mitigation
- Status reporting and stakeholder updates
- Budget tracking and forecasting
- Cross-team coordination

## Project Planning

**Initiation:**

- Define scope and objectives
- Identify stakeholders
- Create project charter
- Establish success criteria

**Planning:**

- Work breakdown structure (WBS)
- Timeline with milestones
- Resource assignment
- Risk register
- Communication plan

**Execution & Monitoring:**

- Track progress vs baseline
- Manage scope changes
- Resolve blockers
- Update forecasts

## Risk Management

| Risk Level | Probability | Impact   | Response                 |
| ---------- | ----------- | -------- | ------------------------ |
| Critical   | High        | High     | Immediate escalation     |
| High       | High/Med    | High/Med | Mitigation plan required |
| Medium     | Med         | Med      | Monitor weekly           |
| Low        | Low         | Low      | Accept and document      |

## Status Report Template

- Project health: Green/Yellow/Red
- Accomplishments this period
- Planned for next period
- Risks and issues
- Decisions needed
- Budget status (spent vs remaining)

## Delivery Checklist

- [ ] All acceptance criteria met
- [ ] Stakeholder sign-off obtained
- [ ] Documentation complete
- [ ] Training delivered
- [ ] Support handoff done
- [ ] Lessons learned captured

## Output

- Project charter and scope document
- Gantt chart / timeline
- Weekly status reports
- Risk register with mitigations
- Change request log
- Project closure report
